# 财务分析与关键指标

> 何时读本文件：需要分析财务报表、计算所有者盈利、评估 ROIC/ROE 质量、或理解透视盈利概念时。

## 核心原则：穿透会计看经济实质

GAAP 财务报表是起点，而非终点。巴菲特始终问：**这家企业真正赚了多少钱？**

---

## 所有者盈利

巴菲特在 1986 年致股东信中提出此概念，作为衡量企业真实盈利能力的最佳工具。

```
所有者盈利 =
  净利润
  + 折旧、折耗与摊销（D&A）
  − 维持竞争地位所需的维护性资本开支
  − 任何必需的营运资金净增加
```

**关键区分——两类资本开支：**

| 类型 | 性质 | 处理 |
|-----|-----|-----|
| 维护性资本开支 | 不花则竞争地位恶化 | 须从所有者盈利中扣除 |
| 成长性资本开支 | 可选，用于扩张新产能 | 不扣除（该部分产生未来回报） |

**为何比 EBITDA 更可靠：**
EBITDA 加回折旧，隐含假设资产不损耗。但资产会老化，维持运营需要真实现金支出。

**估计维护性资本开支的方法：**
- 管理层有时会披露
- 近似：折旧 × 行业经验倍数（轻资产企业常接近 1:1，重资产可能更高）
- 与同业比较

## 关键财务指标

### 盈利能力

**ROE（净资产收益率）**
```
ROE = 净利润 / 平均股东权益
```
- 巴菲特标准：十年均值 >15%，且无过度杠杆
- 须拆解来源：利润率扩张 / 资产周转改善 / 财务杠杆上升
- 靠杠杆抬高的 ROE 是陷阱，非优势

**ROIC（投入资本回报率）**
```
ROIC = 税后净营业利润（NOPAT）/ 投入资本（权益 + 有息负债 − 现金）
```
- 比 ROE 更诚实；剔除资本结构扭曲
- 持续 >15% 为优秀；>20% 为卓越
- **价值创造条件：ROIC > WACC（加权平均资本成本）**

### 现金流质量

**现金转化率**
```
现金转化率 = 经营活动现金流 / 净利润
```
- 持续 >90%：盈利质量高，不依赖会计技巧
- 持续 <70%：需深入调查（可能应收虚增或收入确认激进）

**自由现金流（FCF）**
```
FCF = 经营活动现金流 − 资本开支（合计）
```
注意：含成长性资本开支，更适用于轻资产公司。
高资本开支的成长阶段企业，用所有者盈利视角更公平。

### 资产负债表健康

**债务安全：**
- 债务 / EBITDA < 2 倍：舒适区间
- 利息保障倍数（EBIT / 利息费用）> 5 倍：缓冲充足

**最坏情景测试（必做）：**
假设收入下降 30%——公司能否：
1. 偿还到期债务？
2. 维持核心业务运营？
3. 避免稀释性融资？

### 盈利质量信号

**红旗清单（任一项出现须深挖）：**
- [ ] 应收增速持续高于收入增速
- [ ] 存货增速持续高于收入增速
- [ ] 经营现金流持续低于净利润
- [ ] 频繁使用「调整后盈利」且与 GAAP 差异大
- [ ] 高溢价收购形成大额商誉但回报平庸
- [ ] 频繁关联交易且条款不透明
- [ ] 更换审计师，或审计意见带保留
- [ ] 管理层大量减持

## 透视盈利

适用于：**分析控股公司、投资控股公司、或资产负债表上未并表的大额股权投资**

巴菲特 1989 年首次提出、1990 年首次系统计算，弥补 GAAP 反映被投资企业真实盈利能力的重大缺陷。

**完整计算公式：**
```
透视盈利 =
  伯克希尔报告的经营盈利（剔除资本利得与非经常性项目）
  + 伯克希尔在主要被投资方留存收益中按比例应占份额
  − 若该留存收益以股息形式分给伯克希尔则需缴纳的估计所得税
```

**1990 年实际计算示例（巴菲特原话）：**
> 「取 2.5 亿美元，约为我们 1990 年在被投资方留存经营盈利中的份额；减去 3000 万美元，即若该 2.5 亿美元以股息付给我们我们应多缴的税；将余下 2.2 亿美元加到我们报告的 3.71 亿美元经营盈利上。于是，我们 1990 年『透视盈利』约 5.9 亿美元。」——1990 年巴菲特致股东信

**为何重要：**
持股比例低于 20% 时，GAAP 仅允许确认收到的股息，留存收益从账上「消失」。大都会/ABC 案例：1990 年伯克希尔应占盈利超 8200 万美元，但 GAAP 账面仅约 53 万美元（股息扣税后）。GAAP 严重低估真实盈利能力。

**透视盈利与内在价值增长的联系：**
> 「若伯克希尔内在价值年均增长约 15%，我们的透视盈利也需以大致相同速度增长。」——1989 年巴菲特致股东信

**为何留存收益有时比股息更有价值：**
> 「你必须判断这些未分配盈利对你是否已与已报告盈利同样有价值。我们相信是——事实上可能更有价值。得出『林中鸟可能值手中两只鸟』的结论，是因为我们的被投资方在能干且股东导向的管理者经营下，有时能在自身业务中找到比我们能为资本找到的更优用途。」——1989 年巴菲特致股东信

**GAAP「20% 门槛」的缺陷：**
巴菲特早在 1982 年就批评这一武断门槛：「留存收益对所有股东的价值取决于配置效率，而非你的持股比例大小。」

**实务历史：** 1990–1997 年连续八年计算并公开披露；1998 年收购 General Re 后暂停；2018 年起改为披露被投资方股息与留存收益对比，精神延续。

> 「虽无单一数字能完美概括一切，我们相信透视盈利比 GAAP 数字更准确反映伯克希尔的真实盈利能力。」——1992 年巴菲特致股东信

**分析伯克希尔时必须使用此框架；分析任何持有大额非控股股权的公司时也应考虑。**
